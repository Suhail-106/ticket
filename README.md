# 🎟️ Ticket System (Django)

## 🚀 Setup Instructions

### 1. Clone project
git clone https://github.com/Suhail-106/ticket.git

cd ticket-system

### 2. Install dependencies
pip install -r requirements.txt

### 3. Run migrations
python manage.py migrate

### 4. Generate tickets
python manage.py generate_tickets

### 5. Run server
python manage.py runserver

### 6. Open browser
http://127.0.0.1:8000/

---

## 📸 Features
- QR Code ticket generation
- Scan & redeem system
- One-time use tickets
